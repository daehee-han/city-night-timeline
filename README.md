# city-night-timeline

['불침번'](http://night.20timeline.com)이라는 서비스를 보고 만든 페이지입니다.

21~6시 동안만 타임라인 서비스를 이용할 수 있으며, 이 외의 시간에는 이전 타임라인의 내용들만 검색할 수 있습니다.

## Installation and Settings

```bash
$ pip install -r requirements.txt
$ npm install
$ python manage.py runserver
```
